== Description ==

Простой и удобный плагин для подключения сервиса непорядочных покупателей Blackbox к вашему Woocommerce магазину. Плагин добавит информацию о покупателе в письмо, которое получает администратор сайта. Плагин проверяет покупателя по номеру телефона. Если телефон был найден в базе Blackbox, в письме администратору с деталями заказ вы получите сообщение об этом.

 
== Installation ==

1. Скачайте, установите и активируйте плагин.
2. На сайте blackbox.net.ua найдите свой API ключ для работы с сервисом ( https://blackbox.net.ua/store/#api-panel )
3. В настройках WooCommerce ( WooCommerce->Настройки ) поставьте галочку возле пункта Use Backbox APIю
4. Введите свой API ключ в поле Backbox API key
Все.


= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended
